*PARÂMETROS
.param length = 32nm
.param width = 64nm

* VALIDAÇÃO CIRCUITO DE LEITURA SRAM 6T
.include 32nm_HP.pm

* DECLARAÇÃO DAS FONTES DE TENSÃO
Vvdd vdd gnd 1

* DECLARAÇÃO DO CIRCUITO
* NOMENCLATURA: nome source gate dreno bulk(vdd/gnd) TIPO_PMOS_OU_NMOS w=  l=

* TRANSISTORES DE PASSAGEM
M5 bl sae out vdd PMOS w = width l = length
M6 blb sae outb vdd PMOS w = width l = length

* INVERSOR 1
M3 vdd Qb Q vdd PMOS w = width l = length
M1 Q Qb gnd out_sae NMOS w = width l = length

* INVERSOR 2
M4 vdd Q Qb vdd PMOS w = width l = length
M2 Qb Q gnd out_sae NMOS w = width l = length

* TRANSISTOR DE PASSAGEM GND
M7 out_sae sae gnd gnd NMOS w = width l = length

.end
